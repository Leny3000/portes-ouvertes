Mission SLAM - Développement d'une application

Vous êtes un développeur chargé de créer une application de gestion de stocks.

Objectif de la mission :

Créez une application qui permet de gérer un inventaire de produits pour une entreprise. Vous devez :

  • Ajouter des produits à l'inventaire.
  • Supprimer des produits de l'inventaire.
  • Mettre à jour les informations sur les produits (quantité, prix, etc.).

Une fois votre application développée, vous pourrez tester les différentes fonctionnalités !
